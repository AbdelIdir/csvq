# Control Flow

```
IF condition THEN statements
[ELSEIF condition THEN statements, ...]
{ELSE statements}
END IF;
```

```
WHILE condition DO statements END WHILE;
```

```
WHILE variable, [variable, ...] IN cursor DO statements END WHILE
```

```
CONTINUE
```

```
BREAK
```

```
EXIT
```
